# SwiftUI-P12.5-FriendFace
Hacking With Swift Day 60 Milestone (projects 10-12)

An app that downloads JSON Data from "https://www.hackingwithswift.com/samples/friendface.json" and displays it in a list.

---

List of Features:
- Uses URLSession to download JSON date from "https://www.hackingwithswift.com/samples/friendface.json"
- Uses Codable to convert JSON to Swift types
- Displayes data in a NavigationView, with links to more detailed views
